Screenshots & GIF placeholders

Replace the placeholder files in this folder with real screenshots/GIFs:
- main-ui.png: main window
- merge-manager.png: merge manager dialog
- preview.png: preview dialog
- demo.gif: short demo animation

How to capture on Windows
1) Screenshot: Win+Shift+S → select area → paste into an image editor → save as PNG here.
2) GIF recording: Install ScreenToGif (https://www.screentogif.com/), record and export a small GIF to demo.gif.

Tip
- Keep images lightweight (PNG < 500KB, GIF < 3MB) for fast rendering in README.
